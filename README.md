## About

This is a simple front-end that demos Material-UI. There are two pages available:

  1. Login
  2. Account

The login form is not functional, just click SIGN IN (or click "Account" in the nav bar) to access the account page.

## Features

  * Customized theme colors
  * Show/hide password
  * Responsive drawer on account page (always visible on desktop, tap menu icon to view on mobile)
  * Breadcrumbs
  * Customized auto-complete text input
  * Tooltip on floating action button
  * Table

## Build

To test:
```
npm i
npm start
```

To deploy:
```
npm run build
```
